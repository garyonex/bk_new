# API ecommerce GamesApp
## Estamos creando una api para ser utilizada en una ecommerce de venta de videojuegos

## Development

Open your favorite Terminal and run these commands.

First Tab:

```sh
npm run dev
```

Second Tab:

```sh
npm run start
```

(optional) Third:

```sh
npm run test
```

## Dependencies

```sh
linter
npm i standard

```
```sh
jsonwebtoken
npm i jsonwebtoken

```
